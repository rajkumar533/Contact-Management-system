# Contact-Management-system
Developed a contact management System, which allows users to add, remove, view the contacts. Operations are need to be done by the user to ensure smooth user experience, the complete project was created by using only python code and Python3 version.


#**Explanation:**

#Contact Class:
The Contact class has attributes name, phone, and email.
The __str__ method is overridden to provide a readable string representation of the contact.

#ContactManager Class:
The ContactManager class manages a collection of contacts.

#Methods:
add_contact: Adds a contact to the contact list.
remove_contact: Removes a contact from the list based on the name.
search_contact: Searches for contacts by name.
list_contacts: Lists all contacts in the contact list.

#User Interface:
The main function provides a simple text-based interface for interacting with the contact manager.
Users can add, remove, search for, and list contacts.
Copy the code into a Python file (e.g., contact_management.py).

#How to Run:
Copy the code into a Python file (e.g., contact_management.py).
